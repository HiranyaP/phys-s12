Final Project

Overview: 

My idea for a final project is to create a thrust vectoring rocket. I will be doing this project in a group and we will be making many iterations on the thrust vector control module itself. Thrust is normally generated in a certain direction along the rocket's centerline in conventional rocket engines. Thrust vector control, on the other hand, allows the push to be "vectored" or adjusted to direct the vehicle. This can be accomplished by physically rerouting propellant flow within the engine or by changing the direction of the engine exhaust. Thrust Vector Control can be used in many different ways. Gimballing is a popular technique that involves putting all or part of the engine on a base that can move. This base is called a "gimbal." You can change how hard the engine pushes by moving or turning the gimbal. This lets the rocket change its attitude or direction of flight.

I have always been interested in drones and rockets from a young age and I have always looked to build a thrust vector controlled rocket, the idea of being able to control a rocket almost like a drone fascinates me. 

Idea:

Concept:

The idea is to create a 2 axis gimbal that the rocket engine is mounted to. This 2 axis gimbal would be controlled by servos and it would allow the rocket to correct itself when it is veering off angle. 

We came up with the idea to use compliant mechanisms to be able to control the rocket engine. This would rely on the tension of the plastic to be able to flex and bend and have motion.
